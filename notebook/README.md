# NLP Mini Project 3: Khmer Word Embeddings and Neural Language Modeling

This project implements Khmer word embeddings and neural language modeling using `temples.txt`.

## Main Methods
- Khmer text preprocessing
- Data quality checking
- Plotly EDA
- Skip-gram with negative sampling
- PCA visualization
- N-gram baseline with smoothing
- Neural LM with fixed embeddings
- Neural LM with scratch embeddings

## Run Notebook
```bash
pip install -r requirements.txt
```
Then open the notebook and run all cells.

## Run Streamlit
```bash
streamlit run streamlit_app/app.py
```
